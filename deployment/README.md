# Deployment Overview

This folder documents the installation and setup procedures used to build the SOC environment.

Contents include:
- Virtual machine creation
- Operating system installation
- Wazuh server deployment
- Initial system configuration

Purpose:
To provide reproducible steps for rebuilding the SOC lab from scratch.

SOC Relevance:
Demonstrates the ability to deploy monitoring infrastructure, which is critical for validating detection pipelines and maintaining SIEM environments.
